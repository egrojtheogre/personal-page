# Jorge's Personal Page

A clean, responsive, and modern personal website built to share my professional journey, skills, and projects as a Quality Assurance Engineer.

## Live Project
- GitHub Repository: https://github.com/egrojtheogre/personal-page
- LinkedIn: https://www.linkedin.com/in/jorge-iv%C3%A1n-mart%C3%ADnez-mart%C3%ADnez-a87630b0/

## Project Overview
This website acts as my professional portfolio, highlighting my career evolution from Telecom into QA Automation and Data Analysis. It features:
- **About Me**: A brief introduction to who I am and my professional approach.
- **Experience Timeline**: An interactive, toggleable timeline highlighting my roles (Senior QA Engineer at Caliente Interactive, Software Testing Engineer at EPAM, QA Engineer at Netcracker, etc.).
- **What I'm Learning**: A categorization of my current learning focus (Playwright, Python, Postman, ISTQB) and personal interests (Carpentry, Drumming).
- **Projects**: Showcases of my work, such as E-Commerce E2E test automation and DIY restoration projects.
- **Contact**: Quick links to reach out via email or connect on professional platforms.

## Built With
- **HTML5**: Structured semantic markup.
- **CSS3**: Custom modern styling, including glassmorphism, responsive grids, timelines, and animations.
- **JavaScript (ES6)**: Logic for DOM manipulation and interactive components (timeline toggle).
- **Font Awesome**: Modern iconography.
